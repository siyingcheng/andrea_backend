# Cart

## User Stories

## Anonymous User

As an anonymous user, I can't access the cart, so that I can only browse products without making purchases.

### Registered User

As a registered user, I can:
- Add products to my cart, so that I can prepare items for purchase.
- View the contents of my cart, so that I can review the items I intend to purchase
- Update the quantity of items in my cart, so that I can adjust my order before checkout.
- Remove items from my cart, so that I can modify my order as needed.
- Clear my entire cart, so that I can start a new order if needed.
- Proceed to checkout from my cart, so that I can complete my purchase.

### Admin User

As an admin user, I can view all user carts, so that I can monitor shopping activity on the platform.
