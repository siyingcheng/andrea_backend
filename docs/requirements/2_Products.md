# Products

## User Stories

### Anonymous User

As an anonymous user, I can:
- View detailed product information (name, description, price, availability), so that I can make informed purchasing decisions.
- Search for products using keywords or filters (category, price range), so that I can quickly find items of interest.

### Registered User

As a registered user, I can:
- View detailed product information (name, description, price, availability), so that I can make informed purchasing decisions.
- Search for products using keywords or filters (category, price range), so that I can quickly find items of interest.
- View reviews and ratings left by other users, so that I can gauge the quality of products before making a purchase.
- Leave reviews and ratings for products I have purchased, so that I can share my feedback with other users.
- Update or delete my reviews and ratings, so that I can manage my feedback on products.

### Admin User

As an admin user, I can:
- Add new products to the platform by providing necessary details (name, description, price, stock quantity), so that I can expand the product catalog.
- Edit existing product details, so that I can keep the product information up to date.
- Delete products from the platform, so that I can manage the product catalog effectively.
- View a list of all products, so that I can oversee the available items on the platform.
