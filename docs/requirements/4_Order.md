# Order

## User Stories

### Anonymous User

Anonymous users cannot place orders. They must register and log in to make purchases.

### Registered User

As a registered user, I can:
- Place an order for the items in my cart, so that I can purchase products from the platform.
- View my order history, so that I can keep track of my past purchases.
- View details of a specific order (items, total amount, status), so that I can review my purchase information.
- Cancel an order that has not yet been shipped, so that I can manage my purchases.
- Receive order confirmation via email after placing an order, so that I have a record of my purchase.
- Track the status of my order (processing, shipped, delivered), so that I can stay informed about my delivery.


### Admin User
As an admin user, I can:
- View all orders placed on the platform, so that I can manage and oversee order processing.
- Update the status of an order (processing, shipped, delivered), so that I can keep customers informed about their orders.
