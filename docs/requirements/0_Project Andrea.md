# Project Andrea

This is a demo website for practicing web automation testing frameworks and tools, like Selenium, Playwright, Postman, etc.

This is E-commerce platform for selling products online. The platform provides a user-friendly interface for browsing and purchasing products, as well as features for managing orders and customer accounts.

## Features

1. User Authentication
2. E-Commerce features
   - Product Management
   - Cart Management
   - Order Management
   - Payment Gateway Integration
